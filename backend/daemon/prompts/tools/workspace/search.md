Find files in your workspace and memory directory that match a query. Bare paths resolve under workspace/. Use `memory/...` to search durable memory files. By default uses hybrid ranking (semantic + lexical) so paraphrased queries still find the right file; pass `mode: "lexical"` for an exact substring match. Returns paths, line numbers, and short excerpts. Treat results as discovery only: excerpts are line-level snippets, so almost always follow up with `read` on the top files for full context, then chase any cross-references from there.
