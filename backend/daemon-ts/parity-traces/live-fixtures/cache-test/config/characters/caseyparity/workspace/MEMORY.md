# Memory Index

This file is Casey's map of long-term memory. It is not the memory itself — it is a pointer to the files where memory lives.

## Memory areas

- `memory/log.md` — chronological session log, one line per beat. Read this before continuing a session to remind yourself of the most recent events.
- `memory/people.md` — named NPCs and their current state. Update whenever a named NPC is met, killed, befriended, or moved.
- `memory/places.md` — locations the party has visited, with terse notes on each (atmosphere, threats, unfinished business).
- `memory/setting.md` — current campaign setting and house rules. Read once at the start of a new arc; reference when the player asks about world details.
