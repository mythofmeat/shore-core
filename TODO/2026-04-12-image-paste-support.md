# Feature: Image paste support in shore-tui
shore-tui already has functional image support via the `:image` command that opens a file browser and selects images, but being able to paste images that are copied to the clipboard would also be useful.

Pasting an image should work, essentially, no different to selecting it. Just pressing `ctrl+v` or `ctrl+shift+v` while an image is in the clipboard attaches it to the message.
