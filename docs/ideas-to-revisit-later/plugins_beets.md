# Beets Plugin Idea

Status: idea, not current implementation.

Expose a read-mostly music-library tool backed by Beets. The character could search albums/tracks, notice listening patterns, and talk about the user's library without scraping arbitrary files.

Constraints:

- read-only by default
- no automatic memory writes unless the character explicitly chooses to save a note
- results should be concise enough for tool-loop use
